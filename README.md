## Requirements

- Python 3.x
- Linux

## Usage

### Python :
`python3 modifiedlist.py [ Directory ] [ Days ]`
### Bash :
`./modifiedlist.sh [ Directory ] [ Days ]`

## Example 

`./modifiedlist.sh /home 3`  

`python3 modifiedlist.py /home 3`
